# Landing-Page
This project is a fully responsive Course Landing Page built using HTML, CSS, and basic UI components.
It showcases multiple programming courses (HTML, CSS, JavaScript) with clean card layouts, modern colors, and mobile-friendly design.

The website is designed to help beginners explore and start their coding journey with a simple and attractive interface.
# Features
 Modern, clean UI design using Orange, Black & White theme

 Responsive layout built with media queries

 Course cards with logos (HTML, CSS, JavaScript)

 Smooth button hover effects

 Attractive hero section with CTA button

 Simple navigation bar

 Footer with social media icons
